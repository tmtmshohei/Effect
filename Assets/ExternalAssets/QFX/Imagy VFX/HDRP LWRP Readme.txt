HDRP and LWRP are support, except of the some distortion effects
 
Name of the material: "P_Distortion"
Name of the effects: Rune of Power, Sakura Shield, Void Grenade, Lightning Splash, 
Dark Paralysis, Gunshots, Lightning Blast

Therefore, please just delete/disable particle system with the distortion material.
Insdead of distortion there is simple fresnel effect.

To use default version of the asset in the HDRP/LWRP, you need to upgrade materials: "Edit->Renderer Pipeline->Upgrade Materials"
