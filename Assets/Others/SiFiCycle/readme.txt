SiFi風のマテリアルです。
空間を埋め尽くす回路のような表現をしたいときに使えると思います。
本体は(シェーダーでポリゴンを増やしているので)1ポリゴンなので、ポリゴン数に余裕がなくても比較的気軽にアクセサリーとして使用できます。

【プロパティー】
①BaseColor : 基本の線の色です
②PointColor : 移動する点の色です
③Width : 線の太さです
④PointSize : 移動する点の大きさです
⑤MinAlpha : 透明度の最低値です
⑥Speed : 処理速度です
⑦Cycle : 線の点滅の頻度です
⑧Interval : 線の点滅の間隔です
⑨TesselationFactor : 線の密度です
